HTML Resume Website
This project is a single-page resume website built using only HTML as part of a Web Development assignment.

Features
Clean and readable HTML structure
Sections for About, Skills, Experience, Projects, Education, and Contact
Uses proper semantic HTML tags
No CSS used (as per assignment rules)
How to Run
Download or clone the repository
Open index.html in any modern web browser
